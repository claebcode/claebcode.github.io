Layer Types (BETA)

*This feature is not finished but is a work in progress


  • Local Layers (BLUE)
    • These are your standard layers that you can draw onto.
  • Global Layers (GREEN)
    • A Global Layer will exist on all frames but can have different image data on it.
  • Static Layers (RED)
    • This type of layer can be used as a template. For example, you can create a Static Layer called house, draw the house once, then every time you change the house it will be updated wherever else you use it in your project.
  • Background Layers (YELLOW)
    • These layers will exist on all frames automatically (reduces copy and pasting) and will have the same image data across the project. For example, if you added a line on this layer on the first frame, it would automatically affect all other frames with this layer.
